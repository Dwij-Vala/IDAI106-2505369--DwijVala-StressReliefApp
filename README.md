## Project Overview

This project focuses on designing a Smart Stress Relief App that helps employees manage workplace stress using AI-driven personalization. The solution uses design thinking principles to create a user-centered experience that provides quick and effective relaxation without disrupting workflow.

---

## Objective

The objective of this project is to understand workplace stress and design an innovative solution that delivers instant, personalized, and accessible stress relief using technology.

---

## Scenario Chosen

Build Mindful Work App Prototype for Stress-Free Offices

The app integrates AI, sensory design, and user inputs to create a calming and immersive stress-relief experience through smart pods.

## Design Thinking Process

### 1. Empathize & Define

* Conducted user research with working professionals
* Identified key stress triggers such as workload and deadlines
* Created 10 detailed personas and empathy maps
* Defined a clear problem statement

---

### 2. Ideation

* Brainstormed features based on user needs
* Developed core features like MoodSync AI, Breathe With Me, Pod Scheduler, and Stress Dashboard
* Added innovative features like smart nudges and adaptive environments

---

### 3. Prototype

* Designed a mobile app prototype using Figma
* Created multiple screens including dashboard, AI system, scheduler, and journal
* Focused on clean UI, calming colors, and easy navigation

---

### 4. Testing

* Conducted testing with 8 users
* Collected feedback on usability and experience
* Improved UI clarity, navigation, and feature accessibility

## Reflection

Working on this project helped me understand how important it is to design solutions based on real user needs rather than assumptions. Through research and empathy mapping, I realized that stress is not just a general issue but varies from person to person depending on their role and environment.

One of the key learnings was how design thinking helps break down a problem step by step, making it easier to create meaningful solutions. While designing the prototype, I focused on keeping the interface simple and calming so that users can quickly access features even when they are stressed.

Testing was also a valuable part of the process. It showed me that small design changes, like improving button visibility or simplifying controls, can make a big difference in user experience.

Overall, this project improved my skills in user research, problem-solving, and UI/UX design, and gave me a better understanding of how technology can be used to improve mental well-being in real-world situations.

## Conclusion

The Smart Stress Relief App demonstrates how AI and design thinking can be combined to create a user-centered solution that improves workplace well-being. The prototype provides a practical and innovative approach to managing stress in modern work environments.
